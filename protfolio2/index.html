About

I am Paul Sajo, the creator and founder of Peckwoods , a Social networking platform, currently pursuing my B.tech in Computer Science and Engineering at Christ University, Bangalore, India.

I am highly motivated research oriented senior developer. I also have authored many open-source libraries and tutorials. Besides programming I also have also done quite a bit of 3d modeling using Blender, illustrations using Inkscape, image editing using Gimp and UI designs using Adobe xd. When I am not working on any projects I usually help other developers in Stack overflow community.

I always prefer to work on new and something that I have not worked with before. I usually work alone, but when I am assigned a team I prefer to work in small group with highly motivated people. I am also a believer of "Too many cooks spoil the broth".

Projects

  • Peckwoods

    Peckwoods is a social networking platform where users share their ideas, opinions and thoughts through what's known as peck. Peckwoods also allows users to join an existing community such as Bikes, Music, Food etc, or even create a new community and take the community and its members forward.

    Additionally, users (a.k.a peckers) can send private messages to get in touch with another pecker.

    Read more about Peckwoods in its about page:
    https://peckwoods.pythonanywhere.com/home/#about
    or visit the website
    https://peckwoods.com/

  • Python Designer

    Python designer is a visual scripting tool made to help python programmers easily create classes. A user is expected to add whats called as node from right-click context-menu, which then allows users to add variables and methods. Users can draw connection between nodes to depict inheritance. Once the users is satisfied with the design, user can click on generate to generate the python equivalent code.
